Arrays Problem Set

Create 4 functions using arrays:

  1. printReverse() - takes an array as an argument and prints out the elements in the array in reverse order (don't actually reverse the array itself)
      2. for e.g.,
    • printReverse([1,2,3,4]);
    • //4
    • //3
    • //2
    • //1

  2. isUniform() - takes an array as an argument and returns true if all elements in the array are identical
      3. for e.g.,
    • isUniform([1,1,1,1]); //true
    • isUniform([2,1,1,1]); //false
    • isUniform(["a", "b", "p"]); //false
    • isUniform(["b", "b", "b"]); //true

  3. sumArray() - accepts an array of numbers and returns the sum of all numbers in the array
      4. for e.g.,
    • sumArray([1,2,3]); //6
    • sumArray([10,3,10,4]); //27
    • sumArray([-5,100]); //95

  4. max() - accepts an array of numbers and returns the maximum number in the array
      5. for e.g.,
    • max([1,2,3]); //3
    • max([10,3,10,4]); //10
    • max([-5,100]); //100
